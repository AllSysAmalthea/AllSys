app界面

界面整体描述：
	打开app --> 普通界面
	普通界面 --> 志愿者界面（已注册登陆）or 登陆界面（未登录）
	普通界面 --> 查询人员界面、地图显示界面
	志愿者界面 --> 任务界面
	
 
1、登陆界面
	edittext: Username
	edittext: Password
	button:   Login
	button:   Register

2、注册界面
	edittext: Name
	edittext: Tel
	edittext: ID
	edittext: Password
	edittext: Confirm
	edittext: blood
	edittext: addr
	button:   Done

3、普通界面
	ImageButton: News  （点击进入新闻简报界面）    
	button:   Location   （点击定位）
	button:   Safehouse   （点击查看庇护所信息）
	button:   Survivor   （点击查看幸存者信息）
	button:   Volunteer   (点击进入志愿者界面)

4、志愿者界面
	button: exit
	button: user

5、搜索界面
	EditText: name
	EditText: id
	TextView: result
	button:   search
